Plan Details

Use this page to modify the metadata and policies of a plan. Once you have fully configured the plan, don't forget to lock it: Only locked plans can be used by APIs in the organization.

Policies

The Policies tab allows you to manage the policies for this plan. These policies will be applied whenever a request is made to an API via this plan.

Change Log

The Change Log tab shows a history of all the changes made to the plan. Essentially this is an audit log.
